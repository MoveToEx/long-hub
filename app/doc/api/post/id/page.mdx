## GET /api/post/\[post_id\]/

Returns the information of post specified with `post_id`.

Example request:

```http
GET /api/post/6c753c5e-baca-4f1b-a4ed-952dd402d36c
```

Example response:

```json
{
    "imageURL": "http://localhost:3042/posts/6c753c5e-baca-4f1b-a4ed-952dd402d36c.png",

    "text": "我哭了我装的",

            "id": 229,

```

## PUT /api/post/\[post_id\]/

Modify the metadata of the given post.

Accepts: application/json

Body: New metadata to replace current one. Non-existent fields remain unchanged.  
Response: Post metadata after replacement.

Example request:

```http
PUT /api/post/6c753c5e-baca-4f1b-a4ed-952dd402d36c
Content-Type: application/json

{
    "text": "你妈死了",
    "aggr": 5.0
}
```

Example response:

```json
{
    "imageURL": "https://img.longhub.top/posts/6c753c5e-baca-4f1b-a4ed-952dd402d36c.png",
    "id": "6c753c5e-baca-4f1b-a4ed-952dd402d36c",
    "text": "你妈死了",
```

## DELETE /api/post/\[post_id\]/

Deletes the specified post.

Response: Specified post ID.

Example request:

```http
DELETE /api/post/6c753c5e-baca-4f1b-a4ed-952dd402d36c
```

Example response:

```json
{
    "id": "6c753c5e-baca-4f1b-a4ed-952dd402d36c"
}
```
