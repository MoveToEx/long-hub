
## POST /api/post/similar

Search for posts that are similar to image given.

Formdata:

- `image`: Image to search

Response: A JSON object containing the result.

Example request:

```http
POST /api/post/similar
Content-Type: multipart/form-data; boundary=NMSL

--NMSL
Content-Disposition: form-data; name="image"; filename="image.png"
Content-Type: image/png

< ./image.png
--NMSL--
```

Example response:  

```json
{
    "hash": "1100001111000010010010100000101001101101011010001011011100101001",
    "similar": [
        {
            "id": "fb852b83-71c7-4fb0-ab50-7e5eef044965",
            "imageURL": "http://localhost:3042/posts/fb852b83-71c7-4fb0-ab50-7e5eef044965.jpg",
            "diff": 0
        }
    ]
}
```
