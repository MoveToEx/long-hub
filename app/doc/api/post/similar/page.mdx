## POST /api/post/similar

Search for posts that are similar to image given.

Formdata:

- `image`: Image to upload

Response: A JSON object containing the result.

Example request:

```http
POST /api/post/similar
Content-Type: multipart/form-data; boundary=----NMSL

----NMSL
Content-Disposition: form-data; name="image"; filename="image.png"
Content-Type: image/png

< ./image.png
---NMSL
```

Example response:  

```json
[
    {
        "id": "b1257c16-548c-4876-a31a-d4563df62723",
        "imageURL": "http://localhost:3042/posts/b1257c16-548c-4876-a31a-d4563df62723.png"
    }
]
```
