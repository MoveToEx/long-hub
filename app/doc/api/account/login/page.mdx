## POST /account/login

Authenticates user and leaves session data.

Accepts: `application/json`

Request body:

```json
{
    "username": "<USER>",
    "password": "<PASSWORD>"
}
```

Response: a string indicating login status.  

If succeeds, a `200 OK` will be returned.  
Otherwise, a `401 Unauthorized` will be returned.  

Example request:

```http
POST /api/account/login
Content-Type: application/json

{
    "username": "wcnm",
    "password": "<PASSWORD>"
}
```


Example response:

```json
{
    "id": 1,
    "name": "wcnm",
    "email": null,
    "accessKey": "<KEY>"
}
```
